Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

CSS Syntax

selector {
    property: value;
}

Example:

body {
    background-color: lightblue;
}

CSS Selectors

Selectors are used to target the HTML elements you want to style.

  • Element Selector: Targets elements by their tag name.
    • p { color: red; }
  • Class Selector: Targets elements by their class name.
    • .classname { color: blue; }
  • ID Selector: Targets elements by their ID.
    • #idname { color: green; }

CSS Box Model

The CSS box model describes the rectangular boxes that are generated for elements in the document tree and laid out according to the visual formatting model.

  • Content: The content of the box, where text and images appear.
  • Padding: Clears an area around the content. The padding is transparent.
  • Border: A border that goes around the padding and content.
  • Margin: Clears an area outside the border. The margin is transparent.

CSS Examples

Below are some examples of how CSS can be used:

Text Color

h1 {
    color: blue;
}

Background Color

body {
    background-color: lightgray;
}
